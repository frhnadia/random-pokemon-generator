# Random Pokémon Generator

A simple and interactive web app that fetches a random Pokémon using the **PokéAPI** from https://pokeapi.co/.
Each click generates a new Pokémon with its image of official artwork, types, abilities and dynamic background colors.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- BootStrap 5
- PokéAPI(https://pokeapi.co/)
